# 太空射击游戏
一个只用命令行符号实现要求的大一软件工程期末作业
## 项目描述

开发一款经典的2D太空射击游戏。玩家控制一艘飞船，可以左右移动并发射子弹，击败不断前进并射击的外星人舰队。游戏包含多种敌人类型、关卡推进和分数系统，游戏界面尽量美观~~（感觉并不美观）~~。

虽然只是个期末作业但是做了1个月:(
纯新手，还有很多功能想实现但是没能力实现:(
